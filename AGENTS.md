# AGENTS.md — scaffold-pwa

## Quick start

```bash
npm run dev      # Vite dev server
npm run build    # production build
npm run lint     # ESLint (flat config)
npm run preview  # preview production build
```

No tests, typecheck, or codegen in this repo.

## Stack

React 19 + Vite 6 + React Router 7 — pure JSX (no TypeScript).  
Tailwind 3 classes provided via `src/styles/global.css` (PostCSS pipeline), but most styling uses hand-written CSS utility classes (`.btn`, `.card`, `.sheet`, `.fi`, `.bdg-*`, etc.) defined in that same file.

## RTL + Persian

The app is Farsi-first: `dir="rtl"` on `<html>`, `lang="fa"`, Vazirmatn font loaded from `public/fonts/`.  
Custom CSS assumes RTL. Avoid hardcoding LTR assumptions.

## Architecture

- **Entry:** `src/main.jsx` → mounts `App` inside `BrowserRouter`
- **Contexts** (nested in `src/App.jsx`): Auth → Project → Warehouse → Transfers → Data → Work → Location → Service → Payment → EquipmentType
- **Auth:** JWT in `localStorage('token')`, auto-attached via Axios interceptor. Falls back to hardcoded mock users (`AuthContext.jsx:8-13`) if API is unreachable.
- **Roles:** `Contractor`, `OfficeManager`, `Worker`, `SupportAgent`, `SuperAdmin` (`src/lib/constants.js`)
- **Routes** prefixed by role: `/worker/*`, `/manager/*`, `/admin/*` (`src/routes/AppRoutes.jsx`)
- **API base** from `VITE_API_BASE_URL` env var (`.env`), defaults to `https://localhost:44370/api`

## Offline / PWA notes

- `vite-plugin-pwa` configured with `registerType: 'autoUpdate'`, but `devOptions.enabled: false` (enable for offline dev testing)
- Manifest served from `public/manifest.json` (not generated by plugin — `manifest: false`)
- `src/services/localDB.js`, `syncQueue.js`, `taskSync.js` and `OfflineContext.jsx`, `useOfflineSync.js` are **empty stubs** — IndexedDB (`idb` package) is imported but not wired up

## Conventions

- Component CSS classes from `global.css` (not Tailwind utilities) for UI primitives
- Persian labels in UI (form labels, button text, etc.)
- ESLint flat config in `eslint.config.js` — ignores `dist` only
- `index.html` loads `index.css` (scaffold template leftover — **not** imported by React; the real stylesheet is `src/styles/global.css`)
